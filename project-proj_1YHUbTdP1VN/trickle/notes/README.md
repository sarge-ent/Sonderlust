# Sonderlust - Retro 90s Website

A nostalgic tribute to the golden era of web design, featuring warm colors, skeuomorphic buttons, and typewriter fonts.

## Features

- **Retro Design**: Authentic 90s aesthetic with ridge borders, outset buttons, and warm color palette
- **Skeuomorphic Elements**: Buttons with 3D effects, inset input fields, and embossed panels
- **Typewriter Fonts**: Courier Prime and Special Elite fonts for that classic typing feel
- **Interactive Guestbook**: Sign and view visitor messages
- **Animated Visitor Counter**: Classic incrementing counter display
- **Warm Color Scheme**: Sandy browns, tans, and warm oranges create a cozy atmosphere

## Design Elements

- **Primary Color**: Chocolate (#D2691E)
- **Secondary Color**: Peru (#CD853F)
- **Accent Color**: Sandy Brown (#F4A460)
- **Background**: Cornsilk with gradient to Antique White

## Components

- Header with site title
- Navigation with image buttons (Home, About, Gallery, Blog, Contact)
- Interactive guestbook with form
- Animated visitor counter
- Footer with inspirational message and copyright

## Pages

- **Home (index.html)**: Main page with guestbook and visitor counter
- **Music (music.html)**: Music section
- **Pictures (pictures.html)**: Picture gallery
- **Films and Videos (films.html)**: Film and video collection
- **Art (art.html)**: Art showcase
- **Texts (texts.html)**: Written content and texts
